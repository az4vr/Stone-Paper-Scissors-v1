# Stone-Paper-Scissors-v1
A simple Stone Paper Scissors game built in C++. You can play against the computer, and the game will display the result at the end, showing whether you won, lost, or drew.

## Features

* Play against the computer
* Random computer choices
* Displays the result of each round
* Shows the final game result
* Simple and beginner-friendly C++ project
